Robot

Endpoints de controle direto do Robô. seguem a base http://api_url/robot

GET Help

Retorna a lista de comandos disponíveis.

Tipo de Retorno: JSON

http://api_url/robot/help

GET Position

Retorna a posição atual do Robô em Juntas e em Cartesianos.

Tipo de Retorno: JSON

http://api_url/robot/position

GET Test Position

Move o robô até a posição informada através da url, no modo informado.

Tipo de Retorno: JSON

http://api_url/robot/test_position/:pos_key/:move_type
Parâmetro Descrição
pos_key Chave da posição a ser movida
move_type Modo de movimentação: pode ser 'c' ou 'j' para cartesian ou joints.

GET Remove Piece

Remove uma peça do tabuleiro.

Tipo de Retorno: JSON

http://api_url/robot/remove_piece/:pos_key
Parâmetro Descrição
pos_key Chave da posição a ser removida

GET Place Queen

Coloca uma rainha no tabuleiro.

Tipo de Retorno: JSON

http://api_url/robot/place_queen/:pos_key/:queen_n
Parâmetro Descrição
pos_key Chave da posição a ser colocada
queen_n Número da rainha a ser colocada

GET Capture

Realiza a captura de uma peça.

Tipo de Retorno: JSON

http://api_url/robot/capture/:pos_list
Parâmetro Descrição
pos_list Lista de posições a serem capturadas

GET Detect

Realiza a detecção de peças no tabuleiro.

Tipo de Retorno: JSON

http://api_url/robot/detect

GET Disconnect

Desconecta o Robô.

Tipo de Retorno: JSON

http://api_url/robot/disconnect

Game

Endpoints de controle do jogo. seguem a base http://api_url/game

GET Help

Retorna a lista de comandos disponíveis.

Tipo de Retorno: JSON

http://api_url/game/help

GET Start

Inicia o jogo.

Tipo de Retorno: JSON

http://api_url/game/start/:first_player/:p1_color/:p2_color
Parâmetro Descrição
first_player Primeiro jogador. Int: (0 ou 1)
p1_color Cor das peças do jogador 1. String
p2_color Cor das peças do jogador 2. String

GET State

Retorna o estado atual do jogo.

Tipo de Retorno: Protobuf

http://api_url/game/state

GET Robot Play

Realiza a jogada do robô.

Tipo de Retorno: Protobuf

http://api_url/game/robot_play

GET Move Piece

Movimenta uma peça no tabuleiro.

Tipo de Retorno: JSON

http://api_url/game/move_piece/:origin/:destiny
Parâmetro Descrição
origin Posição de origem da peça
destiny Posição de destino da peça

GET End

Finaliza o jogo.

Tipo de Retorno: JSON

http://api_url/game/end
